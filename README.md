# Global Cyclone Density Map Project 🌪️🌊

## Overview

This project visualizes the global distribution of tropical cyclones using geospatial mapping techniques in QGIS. Historical cyclone track datasets were analyzed to identify regions with frequent cyclone activity.

The project demonstrates spatial visualization of large geospatial datasets and highlights global cyclone patterns across major ocean basins.

## Tools

* QGIS
* Global tropical cyclone datasets (IBTrACS / NOAA)

## Key Steps

* Imported cyclone track data into QGIS
* Processed spatial track datasets
* Generated density maps to identify cyclone hotspots
* Designed final map visualizations for geographic analysis

## Result

Overall, cyclone activity is concentrated near the equator, where atmospheric and oceanic conditions are most favourable for development, though formation does not occur directly on the equatorial line. Despite data and methodological limitations, the analysis demonstrates that global cyclone behavior follows consistent and physically driven spatial patterns.
